## 一、产品概述

### 1.1 产品名称
Lumi-Draw（露米画板）

### 1.2 产品定位
一款 AI 画图微信小程序，接入多种 AI 图像生成模型（如 GPT Image 2、Nano Banana 2 等），让用户零门槛创作 AI 作品。用户满意的作品可发布到社区，形成创作→分享→互动的内容闭环，通过积分制实现商业化变现。

### 1.3 核心价值
- **降低创作门槛**：多模型接入 + 风格模板 + 反推提示词，小白也能出大片
- **社区激励闭环**：一键同款 + 关注体系 + 热门推荐，激发持续创作
- **灵活变现模式**：积分制 + 多渠道获取 + 充值，兼顾免费与付费用户

### 1.4 目标用户
| 用户类型 | 特征 | 核心诉求 |
|---------|------|---------|
| AI 绘画爱好者 | 有一定提示词基础 | 多模型尝试、精细参数控制 |
| 普通创意用户 | 零基础，有表达欲 | 一键玩法、风格模板、快速出图 |
| 社区浏览者 | 以看为主 | 发现优质作品、关注喜欢的创作者 |

## 二、变现体系

### 2.1 积分制概述
- 积分是平台核心虚拟货币，用于 AI 生成消费
- **按模型分档定价**：不同模型单张消耗不同积分，精度/数量作为倍率叠加
- 生成失败自动退还对应积分

### 2.2 积分获取渠道

| 渠道 | 规则 | 说明 |
|------|------|------|
| 新用户赠送 | 注册即送 100 积分 | 一次性，降低首次体验门槛 |
| 每日签到 | 每日签到 +10 积分 | 连续签到里程碑奖励：3天 +20、7天 +50、14天 +100、30天 +300 |
| 签到中断 | 中断后连续天数归零，重新计算 | 无补签机制 |
| 邀请码 | 邀请人 +50，被邀请人 +30 | 新用户首次注册时填写邀请码生效 |
| 邀请防刷 | 每人最多邀请 50 人 | 新用户仅首次填写有效 |
| 充值 | 固定档位 + 自定义金额 | 高档位多送少量积分作为加赠 |
| 会员每日 | 每日领取 50 积分 | 会员订阅期间有效 |

## 三、AI 模型与创作功能

### 3.1 接入模型

| 模型 | 特点 | 标签 | 基础消耗（积分/张·1K） | 精度支持 |
|------|------|------|---------------------|--------|
| GPT Image 2 | 画质细腻·理解力强 | 写实、高清 | 15 | 1K / 2K / 4K |
| Nano Banana 2 | 速度极快·性价比高 | 快速、全能 | 8 | 1K / 2K |
| Nano Banana Pro | 高质量进阶版 | 高质量、细节 | 12 | 1K / 2K / 4K |
| Seedream 4.5 | 独特艺术风格·创意生成 | 艺术、创意 | 10 | 1K / 2K |
| Seedream 5.0 Lite | 最新版·轻量快速 | 快速、最新 | 10 | 1K / 2K |

**精度积分倍率：**

| 精度 | 倍率 | 说明 |
|------|------|------|
| 全高清 1K | ×1 | 标准画质，适合日常创作 |
| 超清 2K | ×1.5 | 高清画质，细节更丰富 |
| 超高清 4K | ×2 | 极致画质，适合精细作品 |

> **实际消耗计算**：基础消耗 × 精度倍率 × 生成数量。例如：GPT Image 2 + 2K精度 + 2张 = 15 × 1.5 × 2 = 45 积分。

**API 接入说明：**
- **平台**：kie.ai（https://kie.ai）
- **文档**：https://docs.kie.ai
- **认证**：Bearer Token（`Authorization: Bearer YOUR_API_KEY`）
- **调用模式**：异步任务制
  - 提交任务：`POST https://api.kie.ai/api/v1/jobs/createTask`
  - 查询状态：`GET https://api.kie.ai/api/v1/jobs/recordInfo?taskId={taskId}`
  - 回调通知：支持 `callBackUrl` 参数，任务完成后自动回调
- **统一接口**：所有模型共用同一端点，通过 `model` 字段区分

### 3.2 创作模式

| 模式 | 触发条件 | 说明 |
|------|---------|------|
| 文生图 | 用户未上传参考图 | 纯文字提示词生成 |
| 图生图 | 用户上传了参考图 | 参考图 + 提示词生成，自动判断无需切换 |
| 反推提示词 | 独立功能入口 | 上传图片→AI 分析→返回描述性提示词，可编辑后带入创作页 |

### 3.3 创作参数

| 参数 | 选项 | 说明 |
|------|------|------|
| 模型选择 | 卡片式横滑，一行三个 | 含模型封面图、名称、优点描述、标签、消耗积分 |
| 提示词 | 大文本卡片，最多 500 字 | 支持上传参考图（图生图），字数计数 |
| 风格类型 | 2×4 网格 + 更多 | 选中后自动将风格关键词追加到提示词末尾发送给模型 |
| 画面比例 | 横滑单行 | 1:1 / 3:4 / 4:3 / 16:9 / 9:16 |
| 图片精度 | 横滑单行 | 全高清 1K（×1）/ 超清 2K（×1.5）/ 超高清 4K（×2） |
| 生成数量 | 横滑单行 | 1~4 张 |

### 3.4 风格类型清单
赛博朋克、赛璐璐、黑白、国风、油画、水彩、二次元、写实、3D、像素、蒸汽波、极简、梦幻、暗黑、复古（共 15+，可通过运营后台动态管理）

### 3.5 生成流程

```
用户点击「开始创作」
  → 显示本次消耗积分预览
  → 扣除积分
  → 提交异步生成任务（多张并行独立请求）
  → 生成中状态（骨架屏动画）
  → 每张独立成功/失败
    → 成功：正常展示
    → 失败：显示失败提示 + 自动退还该张积分
  → 用户可保存图片到相册/草稿箱
  → 用户可一键发布到社区
```

### 3.6 创作结果处理

| 规则 | 说明 |
|------|------|
| 自动存历史 | 生成成功后自动存到云端「我的作品」历史记录 |
| 历史保留期 | 30 天，过期自动清理，节省存储成本 |
| 发布永久保留 | 发布到社区的作品永久保留，需有规则提醒用户 |
| 异步任务 | 用户可离开创作页，回页面时自动检查任务状态并展示结果 |
| 多图容错 | 多张并行请求，每张独立成功/失败，失败退对应积分 |

## 五、详细功能说明

### 5.1 首页

**页面结构（从上到下）：**

1. **Banner 走马灯**
   - 自动轮播（4 秒间隔）+ 触控滑动 + 圆点指示器
   - 内容：活动公告、新模型上线、会员特惠、社区精选等
   - 运营后台可动态配置

2. **热门玩法**
   - 横向滚动卡片：人物美颜、人物证件照、宠物头像、古风国潮、Q 版头像、Logo 设计、壁纸、表情包等
   - 每个卡片含预览图 + 名称
   - 点击 → 进入创作页，自动带入预设提示词和推荐参数（官方预设一键带入）
   - 运营后台可动态管理玩法模板

3. **精选作品**
   - 2 列瀑布流
   - 卡片含：作品图片、作者头像 + 昵称、点赞数
   - 点击 → 进入作品详情页

### 5.2 广场

**页面结构：**

1. **顶栏**
   - 左侧：菜单图标（汉堡）→ 点击展开左侧抽屉
   - 中间：推荐 / 热门 / 最新 三个标签
   - 右侧：搜索图标 → 点击进入搜索页

2. **左侧抽屉**（从左滑出 + 遮罩）
   - 用户信息底栏（头像、昵称、积分）
   - 发布作品（→ 跳转创作页）
   - 我的草稿（→ 跳转草稿箱）
   - 浏览记录（→ 跳转浏览记录页）

3. **次级分类 Tab**
   - 横向滚动：全部 / 二次元 / 风景 / 建筑 / 表情包 / 写实 / 国风 / 人像 / 动物 / 抽象…
   - 支持多选高亮
   - 最右侧下拉角标 → 点击展开下拉筛选面板

4. **下拉筛选面板**
   - 全部风格：多选标签网格
   - 参数筛选：模型 / 尺寸 / 精度
   - 重置 + 确认筛选按钮

5. **作品瀑布流**
   - 2 列，卡片含：图片、作者头像 + 昵称、点赞数
   - 已关注作者的作品有关注标识

### 5.3 搜索页

**页面结构：**

1. **搜索框**（自动聚焦）+ 取消按钮
2. **搜索默认页**（无输入时显示）：
   - 搜索历史（标签云，可清空）
   - 猜你想搜
   - 热门搜索（带排名，前 3 名高亮）
3. **搜索结果页**（有输入时显示）：
   - 实时搜索：匹配作品标题 / 提示词 / 标签
   - 结果以瀑布流展示

### 5.4 创作页（核心页面）

**页面结构（从上到下）：**

1. **模型选择区**
   - 卡片式，一行三个模型卡片，左右滑动
   - 卡片内容：模型封面图、名称、优点描述、标签、消耗积分/张
   - 右侧「全部」入口 → 跳转全部模型选择页
   - 选中状态高亮（蓝色边框 + 阴影）

2. **提示词输入区**（大文本卡片）
   - 左下角 `+` 按钮：上传参考图（上传后自动切换为图生图模式）
   - 上传图片预览卡片：位于输入区左上角
     - 右上角 `✕`：删除参考图
     - 点击预览图：放大全屏查看
   - 右上角内部：字数计数 `0/500`
   - 右下角内部：清空提示词按钮
   - 外部右上角：「反推提示词」入口 → 跳转反推提示词二级页
   - 无参考图 = 文生图，有参考图 = 图生图，自动判断无需手动切换

3. **风格类型选择**（2 行 × 4 列卡片网格）
   - 每个卡片：风格预览图 + 遮罩 + 名称
   - 前 7 个为具体风格，第 8 个为「更多」
   - 「更多」点击 → 从底部弹出全部风格选择抽屉
   - 选中后自动将风格关键词追加到提示词

4. **画面比例选择**（横向滚动单行）
   - 卡片含比例缩略图 + 标签（1:1 / 3:4 / 4:3 / 16:9 / 9:16）

5. **生成数量选择**（横向滚动单行）
   - 卡片含数字 + 「张」（1 / 2 / 3 / 4）

6. **结果展示区**（生成后显示）
   - 生成中：骨架屏动画 + 「AI 创作中...」+ 消耗积分 + 预计时间
   - 已完成：图片网格（2 列），可点击勾选
   - 操作按钮：保存图片 / 发布到社区

7. **底部固定栏**（与 Tab 栏相连）
   - 左侧：「本次消耗 N 积分」
   - 右侧：渐变「开始创作」按钮

### 5.5 反推提示词页（二级页）

1. **上传图片区**（虚线边框，点击选图）
2. **分析按钮**：显示「消耗 5 积分」，点击后显示加载动画
3. **结果区**：
   - 可编辑文本框（AI 返回的描述性提示词）
   - 复制按钮
   - 「带入创作」按钮 → 跳转创作页，自动填入提示词

### 5.6 画廊页（个人主页）

**页面结构：**

1. **个人信息卡**
   - 头像、昵称、个性简介
   - 编辑资料按钮（仅自己可见）

2. **统计卡片**（4 列）
   - 作品数 / 获赞数 / 粉丝数 / 同款数

4. **作品网格** + 管理操作（删除/发布/设为头像等）

**他人视角区别：**
- 只展示已发布作品，无管理权限
- 多一个「关注/已关注」按钮
- 粉丝/关注数可点击进入列表

### 5.7 我的页（账户中心）

1. **用户卡片**：头像、昵称、ID、积分余额（突出显示）
2. **快捷入口四宫格**：充值 / 签到 / 会员 / 邀请
3. **菜单列表**：
   - 消息中心（红点提示未读数）
   - 设置
   - 草稿箱
   - 浏览记录
   - 我的关注
   - 我的粉丝
4. **底部菜单**：联系客服 / 意见反馈

### 5.8 作品详情页

**页面结构（从上到下）：**

1. **大图**（可点击全屏查看）
2. **作者信息栏**：头像 + 昵称 + 作品数/获赞数 + 关注按钮（点击头像/昵称进入他人主页）
3. **作品标题 + 描述**
4. **提示词全文**（带复制按钮）
5. **创作信息**：模型、参数（精度/比例）、风格标签、分类标签

   - 点赞 / 收藏 / 分享 / 举报（图标按钮）

### 5.9 发布流程

**触发入口（多入口）：**
- 创作结果页：生成完成后点击「发布到社区」
- 画廊页：悬浮发布按钮 / 作品管理模式中的「发布」操作
- 左侧菜单抽屉：「发布作品」快捷入口
- 作品详情页（自己的）：管理操作中的「发布」按钮

**发布页（独立二级页）：**
1. 作品预览缩略图（自动携带，支持从草稿箱选择）
2. 标题输入（必填，最多 30 字）
3. 描述文本框（选填，最多 200 字）
4. 标签选择（多选标签云，可选系统预设标签或自定义）
5. **公开提示词开关**（公开后别人可一键同款，隐藏则只展示作品图）
6. 分类选择（单选：二次元/风景/建筑/表情包/写实/国风/人像/动物/抽象等）
7. 永久保留提醒文案：「发布到社区的作品将永久保留，历史记录中的未发布作品 30 天后自动清理。」
8. 发布按钮

**发布审核：**
- 发布前自动过微信内容安全 API（文本 + 图片双重检测）
- 通过则直接上线
- 不通过则提示修改（高亮违规内容）
- 无需人工审核

## 六、用户体系

### 6.1 登录与授权

| 阶段 | 方式 | 说明 |
|------|------|------|
| 静默登录 | wx.cloud 自动获取 openid | 进入小程序即识别身份，创建用户记录 |

| 关键操作 | 微信一键授权手机号 | 生成/发布前需绑定手机号，便于风控和找回 |

### 6.2 新用户引导流程

```
新用户首次进入小程序
  → 静默登录（自动获取 openid，创建用户记录，赠送 100 积分）
  → 引导弹窗（可跳过）：
    → 邀请码填写（选填，填写后双方各得积分奖励）
    → 选择感兴趣的创作方向（选填，用于个性化推荐）
  → 进入首页浏览
  → 首次创作/发布时触发：
    → 手机号绑定弹窗（必填，微信一键授权）
    → 昵称 + 头像授权（必填）
  → 绑定完成后解锁创作/发布功能
```

**引导规则：**
- 邀请码仅首次进入时可填写，跳过后不可补填
- 手机号绑定为创作/发布的强制前置条件，浏览社区无需绑定
- 未绑定手机号的用户在点击「开始创作」或「发布」时弹出绑定提示

## 七、社区治理

### 7.1 内容安全

| 环节 | 机制 |
|------|------|
| 提示词前置过滤 | 生成前调用微信内容安全 API 过滤敏感词，拦截违规内容 |
| 发布自动审核 | 发布前自动过微信内容安全 API，通过则上线，不通过提示修改 |
| 举报功能 | 作品详情页提供举报入口 |

### 7.2 举报处理

| 规则 | 说明 |
|------|------|
| 阈值自动隐藏 | 举报数达到阈值（如 5 次）自动临时隐藏待审 |
| 人工审核 | 管理员在后台审核处理举报队列 |
| 处理结果 | 确认违规则永久删除，误报则恢复 |

### 7.3 AIGC 合规要求

根据《人工智能生成合成内容标识办法》及微信平台相关规定，所有 AI 生成内容必须满足以下合规要求：

| 环节 | 要求 | 说明 |
|------|------|------|
| 显式标识 | 所有 AI 生成图片右下角添加「AI 生成」水印 | 半透明小字水印，不影响观感但清晰可辨 |
| 隐式标识 | 图片文件元数据中嵌入 AIGC 标识信息 | 通过 EXIF/metadata 写入生成模型、时间等信息 |
| 提示词过滤 | 生成前调用微信内容安全 API 过滤敏感词 | 拦截涉政/涉黄/暴力等违规提示词 |
| 生成结果审核 | AI 返回图片后进行内容安全检测 | 调用图片内容安全接口，拦截违规图片 |
| 发布审核 | 发布到社区前再次过内容安全 API | 双重保障，防止绕过生成环节的违规内容 |
| 用户告知 | 创作页明确提示「本作品由 AI 生成」 | 用户知晓并同意 AI 生成内容标识 |

**违规处理：**
- 生成违规内容：直接拦截，退还积分，记录违规日志
- 发布违规内容：拦截发布，提示修改，严重违规可封禁账号
- 多次违规：阶梯处罚（警告 → 临时封禁 → 永久封禁）

## 九、技术架构概述

### 9.1 技术选型

| 层级 | 方案 |
|------|------|
| 前端 | 微信小程序原生开发 |
| 后端 | 纯微信云开发（云函数 + 云数据库 + 云存储） |
| AI 接入 | kie.ai 平台（https://kie.ai）统一 API，Bearer Token 认证，异步任务模式 |
| 支付 | 微信支付（虚拟支付，需企业主体审批） |
| 内容安全 | 微信内容安全 API（文本 msgSecCheck + 图片 mediaCheckAsync） |
| 客服 | 微信官方客服（button open-type=contact） |
| 运营后台 | 小程序内管理员页（识别特定 openid） |

**AI 异步任务架构：**

```
用户点击「开始创作」
  → 云函数「创作服务」：
    1. 校验积分余额
    2. 扣除积分（预扣）
    3. 调用 kie.ai createTask API（携带模型、提示词、比例、精度参数）
    4. 获得 taskId，写入 tasks 集合（状态：pending）
    5. 立即返回 taskId 给前端
  → 前端轮询状态：
    - 每 3 秒调用云函数「任务查询服务」
    - 查询 kie.ai recordInfo API
    - 更新 tasks 集合状态
  → 任务完成（成功/失败）：
    - 成功：下载图片到云存储，更新 works 集合，通知前端展示
    - 失败：更新 tasks 状态为 failed，退还对应积分
  → 备选：callBackUrl 回调模式（云函数作为回调接收器）
```

> **注意**：微信云函数最长执行时间 60 秒，部分模型生成可能超时。采用「提交 + 轮询」分离架构，提交函数只负责创建任务（<5秒），轮询由前端定时触发独立云函数。

### 9.2 核心云函数规划

| 云函数 | 职责 |
|--------|------|
| 用户服务 | 登录注册、资料管理、手机号绑定、新用户引导 |
| 积分服务 | 积分余额查询、消费（预扣）、退还、充值回调 |
| 签到服务 | 每日签到、连续天数计算、里程碑奖励 |
| 邀请服务 | 邀请码生成、验证、奖励发放 |
| 创作服务 | 提示词过滤、调用 kie.ai createTask（异步提交）、结果存储 |
| 任务查询服务 | 查询 kie.ai recordInfo、更新任务状态、处理成功/失败逻辑 |
| 反推服务 | 图片分析、提示词生成 |
| 作品服务 | 发布、查询、详情、删除、编辑 |
| 社区服务 | 点赞、收藏、关注、同款、消息通知 |
| 搜索服务 | 作品搜索、用户搜索 |
| 举报服务 | 举报提交、阈值检查、状态管理 |
| 管理服务 | Banner/模型/风格/玩法模板 CRUD、数据统计 |
| 支付服务 | 充值下单、微信支付回调、积分到账 |
