## 4. 云函数架构

### 4.1 单入口路由模式
```javascript
// cloudfunctions/main/index.js
exports.main = async (event, context) => {
  const { action, data } = event;
  const openid = cloud.getWXContext().OPENID;
  
  switch (action) {
    case 'user.login': return userModule.login(openid, data);
    case 'user.getProfile': return userModule.getProfile(openid);
    case 'ai.generateImage': return aiModule.generateImage(openid, data);
    // ... 其他 action
    default: return { code: -1, msg: '未知操作' };
  }
};
```

// 统一调用格式
function call(action, data = {}) {
  return wx.cloud.callFunction({

    data: { action, data }
  }).then(res => {
    if (res.result.code === 0) return res.result.data;

## 7. 安全策略

### 数据库权限
- **users**：仅创建者和管理员可读写
- **works**：所有人可读，仅创建者可写
- **messages**：仅接收者可读写
- **models/styles/banners/gameplays/categories**：所有人可读，仅管理员可写
- **settings**：所有人可读，仅管理员可写
- **feedbacks/reports**：仅创建者可创建，管理员可读写
- **credits/checkins/invitations/memberships**：仅创建者可读，管理员可读写
- **follows**：所有人可读，仅创建者可写

### 云函数安全
- 所有写操作在云函数中校验 openid
- 管理操作校验 `users.isAdmin`
- 积分操作校验余额是否充足
- AI 生图前校验积分

### 内容安全
- 提示词发布前调用 `security.msgSecCheck`
- 图片发布前调用 `security.imgSecCheck`
- 用户昵称/签名修改时校验
